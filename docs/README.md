# Vent Wall Documentation

This directory contains detailed documentation for each development phase of the Vent Wall project.

## Documentation Structure

- `phase-1-setup.md` - Project setup and database architecture
- `phase-2-ui-components.md` - Core UI components and layout
- `phase-3-posting-system.md` - Posting system and user interactions
- `phase-4-reactions-feed.md` - Reaction system and feed management
- `phase-5-user-profile.md` - User profile and content management
- `phase-6-advanced-features.md` - Advanced features and polish
- `phase-7-testing-deployment.md` - Testing, security, and deployment

## Project Overview

Vent Wall is an anonymous mood-sharing platform where users can post emotional "vents" with mood tags, creating a supportive digital bulletin board for mental health and emotional expression.
